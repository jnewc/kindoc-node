Arithmetic

Arithmetic

Contains a number of utility functions for performing
basic arithmetic.

Example usage:

	var arithmetic = new Arithmetic();
	arithmetic.addition(5, 7); // 12
	arithmetic.division(12, 4); // 3

published properties

firstprop: "somedata",

secondprop: "moredata"

public functions

addition : first, second

Adds the two numbers first and second together and returns the result.

division : numerator, denominator

Divides numerator by denominator and returns the result.
